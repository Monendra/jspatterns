We will be discussing here
1. Objects in javascript
2. Creational design pattern
  -Constructor pattern
  -Module pattern
  -Factories
  -Singletons
3. Structural design pattern
  -Decorator pattern
  -Facade pattern
  -Flyweight
4. Bihavioral design pattern
  -Observer pattern
  -Mediator pattern
  -Command pattern
